# Installation

seldom的安装非常简单。

  • 快速安装

目前已经上传 pypi.org ,可以使用pip命令安装。

> pip install seldom
1
  • 体验最新代码

如果你想随时体验最新的代码，可以使用下面的命令。

> pip install -U git+https://github.com/defnngj/seldom.git@master
1
  • 安装依赖

随着seldom 加入更多的功能，seldom不得不依赖其他的开源库。你可以在 requirements.txt 文件里面看到这些依赖。
